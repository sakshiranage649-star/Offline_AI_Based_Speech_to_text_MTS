# Multilingual Translation: Offline Multilingual AI Translation System

Multilingual Translation is a smart AI-powered translation app that converts text, speech, and documents into multiple languages. It works 100% offline with a simple, fast, and user-friendly interface.

## 🚀 Quick Start

1. **Install Dependencies & Models**:
   - Double-click `install_models.bat`.
   - This will install required Python libraries and download the AI models (~5GB).
   - *Ensure you have an internet connection for this step.*

2. **Run the Application**:
   - Double-click `run.bat`.
   - The application will start and automatically open in your default browser at `http://127.0.0.1:8000`.

## 📂 Project Structure

- **backend/**: FastAPI server and AI logic.
- **frontend/**: HTML/CSS/JS user interface.
- **transly.db**: Local SQLite database for history and user management.
- **Transly_Setup_Guide.docx**: Detailed installation guide for clients.
